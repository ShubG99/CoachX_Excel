# Excel Assignments

These assignments focus on building core Excel skills required for data analysis.

### Assignment 1 – Basic Excel Operations

* Used fundamental formulas such as **SUM, AVERAGE, COUNT, MAX, and MIN**
* Practiced spreadsheet formatting and structured data organization

### Assignment 2 – Logical & Lookup Functions

* Applied **IF statements** for conditional analysis
* Used **VLOOKUP and HLOOKUP** to retrieve data from tables

### Assignment 3 – Data Cleaning

* Cleaned messy datasets using functions like:

  * TRIM
  * PROPER
  * LEFT
  * RIGHT
  * MID
* Standardized and structured raw data

### Assignment 4 – Conditional Formatting & Data Validation

* Implemented **conditional formatting** to highlight key insights
* Used **data validation** to create dropdown-based inputs

### Assignment 5 – Pivot Table Analysis

* Created **pivot tables** to summarize large datasets
* Analyzed metrics such as **sales by region, product category, and salesperson**

### Assignment 6 – Excel Dashboard

* Built an **interactive Excel dashboard**
* Used **pivot charts and slicers** to visualize business performance
